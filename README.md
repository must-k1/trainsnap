# TrainSnap

iPhone向けに考えた、シンプルな電車確認プロトタイプです。現在は大阪の和泉中央駅・南海泉北線に絞り、平日の朝だけ表示する形にしています。

## できること

- 最初に1回だけ駅名と路線を入力
- 次にアプリを開くと、次の電車を大きく表示
- 行き先と遅延も表示
- 3本先まで一覧表示
- 平日の朝だけ表示
- 入力内容はブラウザに保存

## 開き方

1. `index.html` をブラウザで開く
2. 駅名と路線を入力する
3. 保存すると、次の電車表示に切り替わる

## GitHub Pages で公開する手順

1. このフォルダを GitHub のリポジトリに push する
2. GitHub のリポジトリ設定を開く
3. Pages で公開元を `Deploy from a branch` にする
4. ブランチを `main`、フォルダを `/ (root)` にして保存する
5. 表示された URL を iPhone の Safari で開く

## iPhone での使い方

- Safari で開く
- 共有メニューからホーム画面に追加する
- 追加したアイコンから直接開く
